## 📦 Installation

Pick the section that matches your OS below. **Each code block has its own "Copy" button** — copying one will not pull in the other.

<details open>
<summary><b>💻 Kali Linux</b></summary>

```bash
sudo apt update && sudo apt install git python3 python3-pip -y
git clone https://github.com/Q-society/qsoc_scaner.git
cd qsoc_scaner
pip3 install requests colorama
python3 qsoc_scanner.py
```

</details>

<details open>
<summary><b>📱 Termux</b></summary>

```bash
pkg update && pkg upgrade -y
pkg install git python -y
git clone https://github.com/Q-society/qsoc_scaner.git
cd qsoc_scaner
pip install requests colorama
python3 qsoc_scanner.py
```

</details>
